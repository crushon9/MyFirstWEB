JSP (Java Server Page)

Java를 이용한 서버 사이드 템플릿 엔진. jsp 는 자바를 웹서버에서 쉽게 쓰기 위한 기술이며 언어가 아니다
HTML 코드 내에 직접 비즈니스 로직을 삽입할 수 있는 개발 기술이다

EL (Expression Language)

EL은 JSP의 출력 문법을 대체하는 표현 언어
<%= 변수 %> (JSP값표현) → ${ 변수 } (EL값표현)

JSTL (JavaServer Pages Standard Tag Library)

JSTL은 태그를 통해 JSP 코드를 관리하는 라이브러리로서, JSP의 가독성이 좋아집니다.
라이브러리이기 때문에 JSTL을 사용하려면, 라이브러리를 다운로드 해서 추가해야 합니다.
<%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c"%>

Servlet 과 JSP 의 차이점

자바 서블릿은 웹페이지를 동적으로 생성하는 서버 측 프로그램 혹은 그 사양을 말한다
서블릿은 JSP와 비슷한 점이 있지만, JSP가 HTML 문서 안에 Java 코드를 포함하고 있는 반면, 서블릿은 자바 코드 안에 HTML을 포함하고 있다는 차이점이 있다.
