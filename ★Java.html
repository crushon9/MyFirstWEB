Compile & Run

소스코드파일(.java)은 컴파일러(javac.exe)를 통해 컴파일되어 바이트코드파일(.class)이 생성된다. JVM(java.exe)을 통해 실행된다.

byte 1 -2^7 ~ 2^7-1 (-128 ~ 127)
short 2 -2^15 ~ 2^15-1 (-32768 ~ 32767)
int 4 -2^31 ~ 2^31-1 (-2147483648 ~ 2147483647)
long 8 -2^63 ~ 2^63-1 (-9223372036854775808 ~ 9223372036854775807)

Variable

Constant

상수는 선언과 동시에 초기화를 해야하며, 이후 변경이 불가하다.
상수를 선언하는 방법은 변수 앞에 final을 붙여준다

Literal

값 그 자체

String

  • System.out.println('H'); //char
  • System.out.println("H"); //string
  • System.out.println("Hello \"World\""); //escape
  • System.out.println("Hello
    World");     //error
  • System.out.println("Hello "
    + "World");     //ok
  • System.out.println("Hello, [[[name]]] ... bye".replace("[[[name]]]", "Sophie")); //replace
    Number & String 비교
  • System.out.println(6); //number
  • System.out.println("6"); //string
  • System.out.println("6"+"6"); //66
  • System.out.println("6"*"6"); //Error
  • System.out.println("1111".length()); //4
  • System.out.println(1111.length); //Error
