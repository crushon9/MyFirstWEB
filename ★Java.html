Compile & Run

소스코드파일(.java)은 컴파일러(javac.exe)를 통해 컴파일되어 바이트코드파일(.class)이 생성된다. JVM(java.exe)을 통해 실행된다.

byte 1 -2^7 ~ 2^7-1 (-128 ~ 127)
short 2 -2^15 ~ 2^15-1 (-32768 ~ 32767)
int 4 -2^31 ~ 2^31-1 (-2147483648 ~ 2147483647)
long 8 -2^63 ~ 2^63-1 (-9223372036854775808 ~ 9223372036854775807)

Variable

변수란 하나의 값을 저장할 수 있는 메모리 공간이다

Constant

상수는 선언과 동시에 초기화를 해야하며, 이후 변경이 불가하다.
상수를 선언하는 방법 : final int x = 10;

Literal

값 그 자체

Casting

형 변환이란 변수의 타입을 다른 타입으로 변환하는 것이다. 형 변환하고자 하는 변수나 리터널 앞에 변환하고자 하는 타입을 괄호와 함께 붙여주면 된다.

  • 묵시적 형 변환 (자동형변환)
  • 명시적 형 변환 (강제형변환)

Print & Scanner

    Print
  • System.out.println(); 출력 후 개행
  • System.out.print(); 개행없이 출력
  • System.out.printf(); 변수의 값을 지시자 형식으로 변환하여 출력
    Scanner
  • sc.nextInt(); 정수 입력받기
  • sc.next(); 문자열 공백 기준으로 입력받기
  • sc.nextLine(); 문자열 개행 기준으로 입력받기
  • sc.next().charAt(0); 입력받은 문자열중 제일 첫번째 문자 하나만 저장
