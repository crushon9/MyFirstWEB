HTML(Hyper Text Markup Language) :
웹페이지를 만들기 위한 표준 마크업 언어

요소(element)와 속성(attribute)

시작 태그와 종료 태그를 갖는 모든 것들은 요소이고 (예외 a, img, button 등) 모든 요소들은 추가적인 정보를 제공하는 속성을 가질 수 있음
속성은 alt, href, src, style, width, height 등이 있음

줄바꿈과 단락태그


br break line 줄바꿈은 닫는 태그가 없다

p paragraph 단락태그는 닫는 태그가 있다
